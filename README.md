# COMP3421
## Installation

- Clone this repository on your local computer.
- Build docker image.
- Run docker image.

```shell
git clone https://github.com/jimmy12613/COMP3421.git
cd COMP3421/
docker-compose up
```

Container is now ready,\
Network interface via `http://127.0.0.1:8002`\
phpMyAdmin via `http://127.0.0.1:8000`
